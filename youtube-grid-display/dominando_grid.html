Dominando Grid Layouts no CSS

Este tutorial cobre tudo o que você precisa saber para dominar o CSS Grid, desde os conceitos básicos até alinhamentos avançados.

Apresentação e Ferramentas Utilizadas

O CSS Grid Layout é um sistema bidimensional que permite organizar elementos em linhas e colunas de maneira eficiente. As ferramentas principais são:

  • O navegador (Firefox e Chrome têm ótimos DevTools para grids)
  • Um editor de código como VS Code

Ferramenta útil: use o DevTools do Firefox e ative "Exibir linhas do grid" para visualizar a estrutura do layout.

Entendendo os conceitos do Grid Layout

O Grid Layout funciona em dois eixos: linhas e colunas. Você define um container com display: grid; e seus filhos se tornam itens de grid.

div.container {
  display: grid;
}

Nomenclaturas do Grid

Alguns termos importantes:

  • Grid Container: o elemento com display: grid.
  • Grid Items: filhos diretos do container.
  • Grid Lines: linhas que dividem o grid.
  • Grid Tracks: faixas de linhas ou colunas.
  • Grid Cells: unidade entre duas linhas horizontais e verticais.

Iniciando com o CSS Grid

Para começar, defina o container e adicione as colunas e linhas básicas.

.grid {
  display: grid;
  grid-template-columns: 100px 100px 100px;
  grid-template-rows: auto;
}

Grid implícito e explícito

O CSS cria novas faixas automaticamente quando os itens excedem o número de linhas ou colunas definidas — isso é o grid implícito.

.grid {
  grid-template-columns: repeat(3, 1fr);
}

Definindo o tamanho mínimo e máximo das faixas

Use minmax() para definir um tamanho mínimo e máximo para as colunas ou linhas.

.grid {
  grid-template-columns: repeat(3, minmax(150px, 1fr));
}

Alocando os itens do Grid nas posições específicas

Você pode posicionar itens manualmente com grid-column-start, grid-column-end, grid-row-start e grid-row-end.

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 2;
}

Areas do Grid

O grid-template-areas define áreas nomeadas para facilitar o layout.

.grid {
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-columns: 1fr 2fr;
  grid-template-rows: auto 1fr auto;
}

Shorthand grid-template

Permite definir colunas, linhas e áreas em uma única propriedade.

.grid {
  grid-template:
    "header header" auto
    "sidebar main" 1fr
    "footer footer" auto
    / 1fr 3fr;
}

Definindo os espaçamentos dos elementos do Grid

Use gap, row-gap e column-gap para espaçar os elementos.

.grid {
  display: grid;
  gap: 10px 20px;
}

Shorthand grid

Permite criar tanto o template quanto o auto-flow e gaps de uma só vez.

.grid {
  grid: auto-flow dense / repeat(3, 1fr);
}

Alinhando os itens com justify-items e align-items

Essas propriedades alinham os itens dentro de suas células horizontal e verticalmente.

.grid {
  justify-items: center;
  align-items: end;
}

Alinhando os itens da linha e coluna com place-items

Combina justify-items e align-items em uma única linha.

.grid {
  place-items: center end;
}

Alinhando o conteúdo do Grid com justify-content e align-content

Essas propriedades ajustam o alinhamento do grid como um todo dentro do container.

.grid {
  justify-content: space-around;
  align-content: center;
}

Alinhando o conteúdo do Grid horizontalmente e verticalmente com place-content

Combina justify-content e align-content de maneira prática.

.grid {
  place-content: center stretch;
}

Alinhando itens específicos com justify-self e align-self

Permitem ajustar o alinhamento de um item individual dentro da célula.

.item {
  justify-self: start;
  align-self: end;
}

Alinhando itens específicos horizontalmente e verticalmente com place-self

Combina justify-self e align-self em uma sintaxe única.

.item {
  place-self: center stretch;
}
